###### PLETL 第一个命令诞生                                     
1 人类史 第一个PLETL 节点编译机操作 shell脚本 问世 并执行成功                           
指令为：                                 
	                              
PLETL:中节点|表格合并|主码|新增列|ID;                       
                         
完整测试指令如下：                      
                          
节点1                            
获取表名:中医诊断:进行选择;                    
条件为:和:笔记|包含|发热:笔记|包含|身重;                      
获取表列名ID病症;                         
操作:0|行至|30;                    
                           
节点1->2                          
操作:病症|进行分词|词性显色;                 
                          
节点1->3                      
操作:病症|进行分词|DNN;                      
                               
节点（（1->2）+（1->3））->4                     
PLETL:中节点|表格合并|主码|新增列|ID;                      
操作:ID|进行数字排序|从小到大;                      
操作:ID|颜色标记为|红色;                           
                    
输出数据 合并成功。
